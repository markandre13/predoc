PReDoc: Das Editierfenster

[index]

Mit dem Editierfenster können Sie einen zuvor im Dokument markierten Bereich kommentieren.

Dabei können Sie wie folgt vorgehen:

  • Sie wählen im unteren Fenster einen vorgegebenen Korrekturtyp und füllen das dadurch gewählte Formular im Fenster darüber aus.
  • Sie zeichnen mit den Malwerkzeugen eine eigene Erklärung.
  • Sie benutzen einen Korrekturtyp und die Malwerkzeuge.

Das Zeichenfenster

Objekte zusammen verschieben. Wenn Sie während SELECT die [Ctrl] bzw. [Strg] Taste gedrückt halten können Sie mehrere Objekte selektieren und diese dann gemeinsam verschieben.
Text ändern Wenn Sie während SELECT 2× auf die gleich Stelle klicken können Sie einen bereits geschriebenen Text weiter editieren.
