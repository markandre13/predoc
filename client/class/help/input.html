PostScript® PostScript ist ein Druckausgabeformat welches von vielen verbreiteten Textsatz- und Textverarbeitungssystemen wie TeX oder WordPerfect unterstützt wird.
Im Normalfall lassen sich alle Dokument in diesen Typ umwandeln.

Für das Einlesen von PostScript Dokumenten muß ein PReDoc Server installiert worden sein.

GIF Das Graphics Interchange Format ist ein weit verbreitetes Grafikformat. Dadurch ist es unter bestimmten Vorraussetzungen auch möglich, mit Scannern eingelesene Dokumente zu korrigieren.
