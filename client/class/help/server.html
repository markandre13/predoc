PReDoc: Der PReDoc Server

[index]

Zur Installation des Server benötigen sie folgende Software die Sie unter bestimmten Bedingungen kostenlos nutzen können:

  • Ein UNIX® kompatibles Betriebssystem
  • Den GNU C++ Compiler 2.8.1 oder höher
  • Ghostscript 4.0 oder höher

Hardwareseitig bnötigen Sie:

  • Eine Netzwerkverbindung zum PReDoc Client
