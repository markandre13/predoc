PReDoc: Der PReDoc Client

[index]

Zum Betrieb des Java Clients benötigen Sie folgende Systemvorraussetzungen:

  • JAVA 1.1.5 oder höher
  • Eine Netzwerkverbindung
  • Einen über das Netzwerk erreichbaren PReDoc Server

Wenn Sie über keine Netzwerkverbindung verfügen, so müßen Sie den PReDoc Server auf dem selben Rechner wie den PReDoc Client installieren.
Im Anhang wird zudem eine Möglichkeit beschrieben, PReDoc auch ohne einen Server zu betreiben, dabei sollten Sie jedoch über Grundwissen im Umgang mit dem von Ihnen verwendeten Betriebssystem besitzen.
